Commands      ===>  Workings

git --version ===> to Check the current version of github
git commit -m "MESSAGE" ===> Replace your message with MESSAGE
git reset HEAD~         ===> to reset progress after commit
git reset --hard        ===> to reset everything (even deleted files too)
git remove
git branch              ===> to see the branches
git branch "BRANCH"     ===> to make a branch (Replace branch name with 
                             BRANCH)
git checkout "BRANCH"   ===> to switch to the other branch (Replace branch 
                             name with BRANCH)

git merge "BRANCH" -m "MESSAGE" ===> to merge a branch with another or main (Replace branch name & message with BRANCH & MESSAGE)

git pull 
git merge
git fetch
